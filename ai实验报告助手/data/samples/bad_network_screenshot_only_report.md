## 实验目的

通过 Wireshark 观察 TCP 三次握手过程，理解 TCP 建立连接的基本流程。

## 实验环境

- Windows 11
- Wireshark
- 浏览器访问网页

## 实验步骤

1. 打开 Wireshark。
2. 访问一个网页。
3. 找到 TCP 报文。
4. 查看三次握手。

该样例故意保留以下问题，用于测试系统是否能发现计算机网络报告中的典型缺陷：
