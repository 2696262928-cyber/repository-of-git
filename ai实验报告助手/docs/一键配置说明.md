# 一键配置说明

项目根目录提供两个便捷入口：

```text
setup_env.exe
start_app.bat
```

## 一、第一次使用

双击：

```text
setup_env.exe
```

它会自动完成：

1. 检查电脑是否安装 Python。
2. 创建 `.venv` 虚拟环境。
3. 升级 pip。
4. 安装 `requirements.txt` 中的所有依赖。
5. 创建 `config/settings.json`。
6. 验证 Streamlit、PDF 解析、DOCX 解析、OCR 等关键依赖。

## 二、启动系统

环境配置完成后，双击：

```text
start_app.bat
```

或在项目根目录运行：

```powershell
.\.venv\Scripts\streamlit.exe run app.py
```

## 三、DeepSeek API Key

一键配置不会写入 API Key。

如需启用大模型评阅，请手动编辑：

```text
config/settings.json
```

将：

```json
"api_key": "在这里填写你的 DeepSeek API Key"
```

改成自己的真实 DeepSeek API Key。

## 四、注意事项

- 电脑仍需要已安装 Python 3.10+。
- 第一次安装 OCR、OpenCV、ONNXRuntime 等依赖可能需要几分钟。
- `config/settings.json` 不会提交到 Git。
- `.venv` 不会提交到 Git。
