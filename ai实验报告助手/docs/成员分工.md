## Git 协作规范

建议分支：

- `main`：稳定版本
- `dev`：整合开发版本
- `feature/parser`：文档解析功能
- `feature/frontend`：前端页面
- `feature/llm`：大模型接入
- `feature/rubric`：评分规则
- `feature/export`：结果导出

提交信息示例：

```text
feat: add pdf parser
feat: add database rubric
fix: handle empty report text
docs: update project design
test: add parser unit test
```
